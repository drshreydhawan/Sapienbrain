# Recall

One-tap voice second brain: record, transcribe (browser speech recognition), summarize + auto-classify with Claude, task/note/log tracking, WhatsApp assignment, calendar hand-off, morning brief, ask-your-brain.

Static site — no build step. `index.html` is the whole app.

Requires an Anthropic API key, entered once via Settings → set key. The key is stored only in the browser's localStorage on your device; it is never in this repo.
